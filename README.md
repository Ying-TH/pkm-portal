# 🏢 PKM-Portal (PKM Work)

**PKM Work** is an internal corporate program representing the company's digital workspace. Inspired by Nextcloud and integrated with Global UI Standards, it aims to provide a secure and intuitive environment within the company's network.

---

## 🎯 Objectives

- **Centralized Portal:** A unified hub for employee tools and resources.
- **Data Privacy:** Focused on internal operations (On-premise/Private Cloud concept) to prevent data leakage.
- **Development Model:** Serving as a baseline model for Full-stack Web Development using the **Modern Monolith** approach.

## 🛠️ Tech Stack & Scope

- **Backend:** Laravel 11 (PHP 8.2+)
- **Frontend:** Vue.js 3 + Inertia.js (Inertia Protocol)
- **Styling:** Tailwind CSS (Custom Global Scrollbar)
- **Data Simulation:** Mock data via Faker.js & PokeAPI integration
- **Infrastructure:** Docker & WSL:Ubuntu (Environment Parity)

## 📅 Roadmap (Phases)

- [x] **Phase 0-Null:** Project initialization, folder structure, and documentation.
- [ ] **Phase 1 (Current):** Development of Global UI Shell (Topbar, Sidebar, Footer) and Custom Scrollbar.
- [ ] **Phase 2:** Authentication & Role-Based Access Control (Admin/User) using Faker.js for simulation.
- [ ] **Phase 3:** Dynamic Routing, Factory Layout mapping, and full navigation implementation.
